# Cafe Zen Beats

A minimal yet stylish e-commerce web app for selling beats by producer Cafe Zen.

## Features

- **Modern UI Design**: Clean, minimal interface with smooth animations
- **Beat Browser**: Browse beats by genre with filtering options
- **Audio Player**: Built-in audio player with controls
- **Shopping Cart**: Add beats to cart with live updates
- **Responsive Design**: Works perfectly on all devices
- **Component Architecture**: Reusable components for maintainability

## Project Structure

```
cafe-zen-beats/
├── index.html              # Main HTML file
├── components/             # Reusable components
│   ├── header.html         # Navigation header
│   └── footer.html         # Footer with links
├── styles/                 # CSS files
│   ├── main.css           # Base styles and design system
│   └── components.css     # Component-specific styles
├── js/                     # JavaScript files
│   └── main.js            # Main application logic
├── assets/                 # Static assets
│   ├── images/            # Beat cover images
│   └── audio/             # Audio files
└── pages/                  # Additional page templates
```

## Design System

### Colors
- Primary: #2c3e50 (Dark blue-gray)
- Accent: #3498db (Blue)
- Background: #ffffff (White)
- Surface: #f8f9fa (Light gray)

### Typography
- Font: Inter (system fallbacks)
- Clean, minimal font stack for optimal readability

### Components
- **Header**: Fixed navigation with cart and search
- **Beat Cards**: Preview, add to cart, metadata
- **Audio Player**: Fixed bottom player with controls
- **Cart Sidebar**: Slide-out cart with item management
- **Footer**: Links, social media, newsletter

## Getting Started

1. Open `index.html` in your browser
2. Browse the featured beats
3. Click play to preview beats
4. Add beats to cart
5. View cart and checkout
